# Workflow: Frontend Geliştirme

Bu workflow, tasarım planını çalışan bir web arayüzüne dönüştürür.

## Adımlar

1. `index.html` yapısını oluştur (Semantic HTML).
2. `style.css` ile modern tasarımı uygula (Glassmorphism, Vibrant Colors).
3. `app.js` ile mantığı kodla.
4. Dosyaları hedef dizine kaydet.
