# Alloy AI

Premium AI Agent with **Alloyty Pipeline** for VS Code.

## Features

- **Alloyty Pipeline**: A 18-agent orchestrated development flow.
- **RARV Telemetry**: Real-time Reason-Act-Reflect-Verify status streaming.
- **HITL Bridge**: Secure Human-in-the-Loop approval system for all terminal and file actions.
- **Hardened Security**: Integrated Path Traversal and Command Injection protection.
- **Rollback Integrity**: Automated Git-based and Shadow Copy checkpoints.

## Installation

Launch VS Code and open the Extension Activity Bar, search for **Alloy AI**.

## Usage

1. Open a workspace folder.
2. Click the **Alloy AI** icon in the sidebar.
3. Enter your task in the **Mission Control** panel.
4. Review and approve agent actions as they happen.

## Security

Alloy AI is designed with a "Security-First" approach, utilizing recursive `realpath` verification and shell-less execution for all binary interactions.
